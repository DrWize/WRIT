Your Kindle silently saves every word you look up into a file called vocab.db. This guide walks you through extracting it and converting it into the enriched Excel file that WRIT can load.

🔌
Step 1 — Connect your Kindle via USB
Plug your Kindle into your computer using a USB cable. It will appear as a drive (like a USB stick) in your file explorer.
Windows: appears in "This PC" · Mac: appears on Desktop or in Finder sidebar
🔍
Step 2 — Find vocab.db
Inside the Kindle drive, the file lives at system › vocabulary › vocab.db. The system folder is hidden by default — use your OS search to find it quickly:
Windows: Click in the search bar at the top of File Explorer and type vocab.db. Hit Enter.
Mac: Open Finder, select the Kindle drive in the sidebar, then press ⌘ + F and search for vocab.db.
Once found, copy the file to your Desktop (don't move it — leave the original on your Kindle).
🌐
Step 3 — Open the export tool
Go to kindle-vocab-export.vercel.app — a free tool by Henry Wang that reads your vocab.db and produces a clean Excel file with word, definition, and lookup count.
Open Export Tool ↗
📂
Step 4 — Upload vocab.db to the tool
On the export tool page, click the upload area (or drag and drop) and select the vocab.db file you copied to your Desktop. The tool reads the file entirely in your browser — nothing is uploaded to any server.
⬇️
Step 5 — Download the Excel file
After the tool processes your database, click "Download Excel" (or equivalent button). You'll receive a .xlsx file with columns for word, usage context, and lookup frequency.
🎮
Step 6 — Load it into WRIT
Drag the downloaded .xlsx file onto the upload zone above, or click the zone to browse for it. WRIT will read the word and definition columns automatically and you can start playing straight away.
💡 Tip: Words looked up only once are probably not worth drilling — the export tool shows lookup counts, so you can delete low-frequency rows in Excel before loading to keep your set tight.
